# proyecto-CDI
CDI project repository: Franco Feo, Victor Romero, Alberto Santiago

Alberto Santiago, app "Where is my car":
La app muestra los siguientes elementos:
1-Switch "Activa ubicación".
2-Mapa localización actual.
3-Botón "Guarda tu localización".
4-Botón "Haz click para ver dónde aparcaste".
5-Mapa localización guardada.

Uso: Al activar el switch "Activa ubicación" se muestra correctamente la latitud y longitud numérica bajo el switch, y en el mapa se muestra un marker de color rojo que apunta a la ubicación actual. Tras ello pulsamos el botón "Guarda tu localización", dichas coordenadas serán guardadas en tinyDB (En local), y se mostrará un mensaje emergente de que la localización se ha guardado exitosamente. Si pulsamos el botón "Haz click para ver dónde aparcaste", se muestra en el mapa 2 el marcador azul, apuntando a la dirección que se guardo anteriormente.
